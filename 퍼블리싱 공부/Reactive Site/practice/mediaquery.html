미디어 쿼리

미디어 쿼리는 화면 크기에 따른 각각의 속성 값을 지정하여, 여러가지 화면을 구성하는 기술이다.

@media only all (조건문) 실행문

  • @media : 미디어 쿼리가 시작됨을 표시
  • only : 미디어 쿼리구문을 해석하라는 명령어 (생략가능)
  • all : 미디어 쿼리를 해석해야 할 대상을 나태냄 (생략가능)
    • all : 모든 미디어 유형에서 사용할 CSS를 정의
    • print : 인쇄 장치에서 사용할 CSS를 정의
    • screen : 컴퓨터 스크린에서 사용할 CSS를 정의
    • aural : 화면을 읽어 소리로 출력하는 장치에서 CSS를 정의
    • tv : TV에서 사용할 CSS를 정의
    • handheld : 손으로 들고 다니는 장치를 사용할 때 CSS를 정의
    • projection : 프로젝터를 위한 CSS를 정의
  • and : 앞에 있는 조건과 나타냄 (생략가능)
  • (조건문) : 해당 조건을 설정 할 수 있다.
  • {실행문} : 조건에 따른 실행 명령을 설정
